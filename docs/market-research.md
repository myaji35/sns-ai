# SNS-AI 시장 조사 (Market Research)

## Executive Summary

SNS-AI가 타겟하는 한국 소상공인/중소기업 SNS 마케팅 시장은 **잠재 시장 규모 약 7,700억 원** (연간 기준)으로 추정되며, AI 콘텐츠 생성 기술의 급속한 발전과 함께 고성장이 예상됩니다. 771만 개 중소기업/소상공인 중 약 30%만 효과적으로 SNS를 활용하고 있어, 나머지 70%가 잠재 고객층입니다.

### 1.2 시장 규모 추정

#### 접근 방법 1: 마케팅 예산 기반 (Bottom-up)

**가정:**
- 타겟 사업체: 450만 개 (SNS 중요도 높은 업종)
- SNS 마케팅 예산: 평균 월 17만 원 (연 200만 원)
  - 소상공인 기준: 월 70만 원~100만 원의 마케팅 예산 중 SNS 비중 20-30%
  - 중소기업 기준: 월 1,000만~3,000만 원 중 SNS 비중 10-20%

**계산:**
```
450만 개 × 20% (실제 SNS 투자) × 17만 원/월 × 12개월
= 1,836억 원/년
```

#### 접근 방법 2: SNS-AI 도입 가능 시장 (TAM/SAM/SOM)

**TAM (Total Addressable Market) - 전체 잠재 시장:**
```
771만 개 전체 기업 × 5만 원/월 (평균 SNS-AI 구독료) × 12개월
= 약 4조 6,260억 원/년
```

**SAM (Serviceable Available Market) - 실질적 접근 가능 시장:**
```
450만 개 (SNS 중요 업종) × 30% (디지털 마케팅 적극 활용) × 5만 원/월 × 12개월
= 약 8,100억 원/년
```

**SOM (Serviceable Obtainable Market) - 실제 확보 가능 시장 (초기 3년):**
```
연도 1: 1,000개 사업체 × 3.5만 원/월 × 12개월 = 4.2억 원
연도 2: 5,000개 사업체 × 4만 원/월 × 12개월 = 24억 원
연도 3: 10,000개 사업체 × 5만 원/월 × 12개월 = 60억 원
```

**결론: 현실적 타겟 시장 규모 = 약 7,700억~8,100억 원/년 (SAM 기준)**

- **연령**: 25-45세 (디지털 네이티브)
- **위치**: 서울/경기 상권 밀집 지역
- **페인 포인트**:
  - 매일 SNS에 올릴 콘텐츠 생각하는 것이 스트레스
  - 사진은 찍지만 캡션 작성이 어려움
  - Instagram, 네이버 플레이스, 카카오맵 등 다중 플랫폼 관리 번거로움

#### 시장 규모
- **추정 사업체 수**: 약 100만 개
- **디지털 적극 활용**: 30% = 30만 개
- **초기 3년 확보 목표**: 10,000개 (3.3%)

- **잠재 고객**: 6만 개 (30%)

## 5. 경쟁 강도 분석 (Competitive Intensity)

### 5.1 Porter's Five Forces

#### 1. 신규 진입 위협 (Medium-High)
- **기술 진입 장벽**: 중간 (LLM API 활용 시 낮음)
- **자본 진입 장벽**: 낮음 (클라우드 기반 SaaS)
- **브랜드 충성도**: 낮음 (시장 형성 초기)
- **완화 전략**: 빠른 시장 선점, 네트워크 효과 구축 (더 많은 데이터 → 더 나은 AI)

#### 2. 공급자 협상력 (Medium)
- **주요 공급자**: OpenAI, Anthropic, AWS/GCP
- **전환 비용**: 중간
- **완화 전략**: 멀티 LLM 전략, 오픈소스 모델 병행

#### 3. 구매자 협상력 (High)
- **가격 민감도**: 매우 높음 (소상공인)
- **전환 비용**: 낮음
- **완화 전략**: Sticky 기능 (콘텐츠 히스토리, 브랜드 학습 데이터), 우수한 고객 경험

#### 4. 대체재 위협 (Medium-High)
- **대체재**: DIY (ChatGPT 직접 사용), 외주 업체, 직원 채용
- **완화 전략**: 올인원 편의성, 워크플로우 자동화, ROI 명확화

#### 5. 기존 경쟁자 (Medium)
- **직접 경쟁자**: 거의 없음 (한국 소상공인 특화 AI+SNS 솔루션)
- **간접 경쟁자**: Hootsuite, Buffer, Jasper 등 (글로벌, 고가)
- **완화 전략**: 로컬 강점 (한국 SNS, 언어, 문화)

**종합 평가**: 시장 진입 타이밍 유리, 단 빠른 실행 필요
